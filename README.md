**Advanced Calculator** is a fork of [Graph89](https://github.com/eanema/graph89), an open-source calculator emulator. Although I did not use the original version of Graph89, it can be found [here](https://bitbucket.org/dhashoandroid/graph89-paid/). 

Unlike Graph89, this project includes the [PedroM](http://www.ticalc.org/archives/files/fileinfo/319/31951.html) ROM image by default, so it can be used instantly with no setup required after installing. Like Graph89, other ROM images can be installed as well. 

Besides adding the PedroM ROM image, I added 16 KB native support for newer devices. I also changed the icon, the title, and the package name.
